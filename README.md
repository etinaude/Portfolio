## 🧩 Prerequists

Install [Node.js](https://nodejs.org/en/)

Install the dependencies

```zsh
npm i
```


## 🏃 Run

Run the follwoing command in yoir terminal:

```zsh
npm run dev
```

and open [localhost:3000](https://localhost:3000)
